## Logging files/folders

To check the second task module, you must:
1. Start the project.
2. Stop the project.
3. The log file will be created in bin\Debug\netcoreapp3.0\Logs.

3. To run the file Report Generator.bat which be located in the root of Module9/LogParser/
4. The finished report will be located in the root of Module9/LogParser/
5. The file name is log_report. {timestamp}. txt
